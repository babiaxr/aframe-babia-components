City of 2020

Each building represents a file of the code, each quarter and the quarter level represents the folder tree that file belongs to, each building has 3 metrics

Metrics:
Area: number of functions. Height: LoC per function. Color: Tokens

Interaction:
Click/Over on a building to see the metrics, click on a quarter to see the folder
